hkDeactivatorType

Bethesda Havok, based on hkpRigidBodyDeactivator::DeactivatorType.
Deactivator Type determines which mechanism Havok will use to classify the body as deactivated.

Choices

Number Name Description
0 DEACTIVATOR_INVALID Invalid
1 DEACTIVATOR_NEVER This will force the rigid body to never deactivate.
2 DEACTIVATOR_SPATIAL Tells Havok to use a spatial deactivation scheme. This makes use of high and low frequencies of positional motion to determine when deactivation should occur.
