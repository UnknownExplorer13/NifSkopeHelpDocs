BSPartFlag

Editor flags for the Body Partitions.

Choices

Number Name Description
0x00000001 PF_EDITOR_VISIBLE Visible in Editor
0x00000100 PF_START_NET_BONESET Start a new shared boneset. It is expected this BoneSet and the following sets in the Skin Partition will have the same bones.
