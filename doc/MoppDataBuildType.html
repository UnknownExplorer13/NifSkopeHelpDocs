MoppDataBuildType

Bethesda Havok.
A byte describing if MOPP Data is organized into chunks (PS3) or not (PC)

Choices

Number Name Description
0 BUILT_WITH_CHUNK_SUBDIVISION Organized in chunks for PS3.
1 BUILT_WITHOUT_CHUNK_SUBDIVISION Not organized in chunks for PC. (Default)
2 BUILD_NOT_SET Build type not set yet.
