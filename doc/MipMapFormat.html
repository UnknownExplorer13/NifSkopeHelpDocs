MipMapFormat

Describes how mipmaps are handled in an NiTexture.

Choices

Number Name Description
0 MIP_FMT_NO Texture does not use mip maps.
1 MIP_FMT_YES Texture uses mip maps.
2 MIP_FMT_DEFAULT Use default setting.

Storage Type

uint
